# [GitHub Release][]

[![Version badge][]][version]
[![Build badge][]][build]

GitHub Release is a command-line utility for uploading files to GitHub
releases.

``` sh
github-release upload \
  --token '...' \
  --owner 'someone' \
  --repo 'something' \
  --tag '1.2.3' \
  --file 'path/to/example.tgz' \
  --name 'example-1.2.3.tgz'
```
